## Git on Windows (branch checkout)

If you see "Unlink of file … failed" or "Deletion of directory … failed" when running `git checkout`, see **[docs/WINDOWS_GIT.md](docs/WINDOWS_GIT.md)** for causes and a safe workflow. You can also use **`.\scripts\safe-checkout.ps1 -Branch <name>`** from repo root.
